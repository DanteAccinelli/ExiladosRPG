# Exilados RPG Class Architecture

---

1. Core
(Game loop, input logging, commands, global state)

1.1 Game Loop / State

### GameManager
  - StateManager StateManager, InputHandler InputHandler, bool IsRunning
  - StartGame(), Update(), ExitGame(), InitializeSystem(), GameLoop()

### StateManager
  - GameState CurrentState
  - ChangeState(GameState new State), GetCurrentState(), UpdateState()

### GameState
  - GameState(enum) -> MainMenu, Playing, Combat, Inventory, Dialogue, GameOver

1.2 Commands & Input

### ICommand
  - string Name, string Description
  - Execute()

### CommandParser
  - Parse(string input), ExecuteCommand(string input), RegisterCommand(string key, ICommand command)

### CommandHelpSystem : ICommand
  - string Name, string Description, CommandParser parser
  - Execute(), ShowAllCommands()

### IInput
  - ReadLine(), ReadInt()

### ConsoleInput : IInput
  - ReadLine(), ReadInt()

//More Input types

### InputHandler
  - IInput Input, CommandParser parser, StateManager StateManager
  - HandleInput() 

1.3 Logging

### ILogger
  - LogInfo(string message), LogWarning(string message), LogError(string message)

### Logger : ILogger
  - bool DebugMode
  - LogInfo(string message), LogWarning(string message), LogError(string message)

2.1 Base Interfaces

### IDamageable

### IAttacker

### IHealable
  - Heal(int amount)

2.2 Stats & Effects

### Stats
  - int MaxHP, int HP, int Attack, int Defense, int Speed, int CritChance, int CritDamage

### StatusEffectContainer
  - List<IStatusEffect> ActiveEffects
  - Add(IStatusEffect effect), Remove(IStatusEffect effect), UpdateTurnStart(Character c), UpdateTurnEnd(Character c)

### IStatusEffect
  - String Name {get;} int Duration {get;} bool IsExpired{get;}
  - OnApply(Character target), OnTurnStart(Character target), OnTurnEnd(Character target), OnExpire(Character target)

### PoisonEffect : IStatusEffect
  - string Name, int Duration, bool IsExpired, int DamagePerTurn
  - PoisonEffect(int damage, int duration)

//More Effect types

### Player : Character
  - Inventory Inventory, EquipmentManager EquipmentManager, List<IAbility> Abilities, SkillTree SkillTree, int Gold, int Experience
  - GainExperience(int amount), LevelUp(), UnlockAbility(IAbility ability), EquipItem(Item item), UnequipItem(Item item)

### Enemy : Character
  - LootTable LootTable, IAIBehavior AI

### MeleeEnemy : Enemy

### CombatSystem

### TurnOrderManager
  - List<Character> TurnOrder
  - Initialize(List<Character> actions), Character GetNextAction(), RefreshTurnOrder()

3.2 Combat Actions

### ICombatAction
  - Execute(GameContext context)

### AttackAction : ICombatAction
  - Character Attacker, Character Defender
  - Execute(GameContext context)

### UseItemAction : ICombatAction
  - Player User, Item Item
  - Execute(GameContext context)

### AbilityAction : ICombatAction
  - Character User, IAbility Ability
  - Execute(GameContext context)

//More Action types

### DamageFormula
  - CalculateDamage(Character attacker, Character defender), bool RollCritical(Character attacker)

### StatusEffectProcessor
