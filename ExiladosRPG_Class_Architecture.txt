# Exilados RPG Class Architecture

---

## Core

### GameManager
  - StateManager stateManager, InputHandler inputHandler, bool IsRunning
  - StartGame(), Update(), ExitGame(), InitializeSystem(), GameLoop()

### StateManager
  - GameState currentState
  - ChangeState(GameState new State), GetCurrentState(), UpdateState()

### GameState
  - GameState(enum) -> MainMenu, Playing, Combat, Inventory, Dialogue, GameOver

### ICommand
  - string Name, string Description
  - Execute()

### CommandParser : ICommand
  - string Name, string Description
  - Parse(string input), ExecuteCommand(string input), RegisterCommand(string key, ICommand command)

### CommandHelpSystem : ICommand
  - string Name, string Description
  - ShowAllCommands(), ShowCommandDetails(string command), ShowCommandsByState(GameState state)

### ILogger
  - LogInfo(string message), LogWarning(string message), LogError(string message)

### Logger : ILogger
  - bool DebugMode
  - LogInfo(string message), LogWarning(string message), LogError(string message)

## Characters

### IDamageable
  - TakeDamage(int amount), IsAlive()

### IAttacker
  - Attack(IDamageable target)

### IHealeable
  - Heal(int amount)

### Character: IDamageable, IAttacker, IHealable
  - string Name, int HP, int MaxHP, int Attack, int CritChance, int Defense, int Speed, int Level, List<IStatusEffect> StatusEffects
  - Attack(Character target), TakeDamage(int amount), Heal(int amount), ApplyStatusEffect(IStatusEffect effect), RemoveStatusEffect(IStatusEffect effect), IsAlive()

### Player : Character
  - Inventory Inventory, EquipmentManager EquipmentManager, List<IAbility> Abilities, SkillTree SkillTree, int Gold, int Experience
  - GainExperience(int amount), LevelUp(), UnlockAbility(IAbility ability), EquipItem(Item item), UnequipItem(Item item)

### Enemy : Character
  - LootTable LootTable, IAIBehavior AI
  - PerformAction(Player player), DropLoot(), InitializeStats()

### IAIBehavior
  - DecideAction(Enemy enemy, Player player)

  - DecideAction(Enemy enemy, Player player)

## Combat

### CombatSystem
  - Player player, Enemy enemy, bool IsCombatActive
  - StartCombat(Player player, Enemy enemy), PlayerTurn(), EnemyTurn(), EndCombat(), CheckVictory()

### DamageCalculator
  - CalculateDamage(Character attacker, Character defender), ApplyModifiers(int baseDamage, List<StatusEffect> effects)

### IStatusEffect
  - String Name {get;} int Duration {get;} bool IsExpired{get;}
  - OnApply(Character target), OnTurnStart(Character target), OnTurnEnd(Character target), OnExpire(Character target)

### PoisonEffect : IStatusEffect
  - string Name, int Duration, bool IsExpired, int DamagePerTurn
  - PoisonEffect(int damage, int duration)

  - GenerateRooms(), ConnectRooms(), AssignRoomTypes()

### IRoom
  - int RoomID, List<Enemy> Enemies, List<Item> Items, List<Trap> Traps, bool Visited
  - Enter(Player player), TriggerEvent(), ClearRoom()

## Items

### IEquipable

  - Apply(Item item)

## Systems

### IInventory
  - AddItem(Item item), RemoveItem(Item item), List<Item> GetItems()

### Inventory : IInventory
  - List<Item> Items
  - AddItem(Item item), RemoveItem(Item item), UseItem(Item item), ListItems()

### EquipmentManager
  - Item Weapon, Item Armor, Item Acessory
  - EquipItem(Item item), UnequipItem(Item item), GetEquippedStats()

### IAbility
  - string Name
  - Activate(Player player)

### Ability : IAbility
  - string Name, int Cooldown
  - Activate(Player player), ApllyEffect()

### SkillTree
  - List<SkillNode> Nodes
  - UnlockSkill(SkillNode node), GetAvailableSkills()

### SkillNode
  - string Name, bool IsUnlocked

### IShop
  - List<Item> GetInventory(), Buy(Player player, Item item), Sell(Player player, Item item)

### EconomySystem : Ishop
  - int PlayerGold
  - BuyItem(Item item), SellItem(Item item), GenerateShopInventory(), GetInventory()

### ISaveable
  - Serialize(), Deserialize(string data)

### SaveSystem
  - SaveGame(), LoadGame(), AutoSave()

### ProfileManager
  - List<string> Profiles
  - CreateProfile(string name), LoadProfile(string name), DeleteProfile(string name)

### IDataLoader
  - Load()

### IGameEvent
  - Execute(Player player)

### TreasureEvent : IGameEvent
  - Execute(Player player)

### DataManager : IDataLoader
  - LoadItems(), LoadEnemies(), LoadEvents(), Load()


### EventSystem
  - TriggerEvent(Room room), ResolveEvent()

### DialogueSystem
  - StartDialogue(string text), PresentChoices(List<string> choices), ResolveChoice(int choice)

### FactionSystem
  - Dictionary<string, int> Reputation
  - ModifyReputation(string faction, int value), GetFactionStanding(string faction)

### AchievementSystem
  - List<string> UnlockedAchievements
  - UnlockAchievement(string achievement), CheckConditions()

### MetaProgressionSystem
  - List<Relic> UnlockedRelics
  - UnlockRelic(Relic relic), UnlockAbility(Ability ability)

### MetricsSystem
  - int Deaths, int Kills, float RunTime
  - RecordDeath(), RecordKill(), RecordRunTime(float time)
